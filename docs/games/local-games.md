# Lokale Spiele

Lokale Spiele sind eine spezielle Art von ZWOO-Spielen, die ohne Internetverbindung in einem lokalen Netzwerk gespielt werden können.

::: warning Achtung
Lokale Spiele benötigen einen Host-Computer mit der installierten und laufenden ZWOO-Desktop-App. Der Host-Computer muss mit demselben Netzwerk wie die anderen Spieler verbunden sein.
:::

## Einen lokalen Server starten

Um einen lokalen Server zu starten, müssen Sie die ZWOO-Desktop-App auf Ihrem Computer installieren und ausführen. Sobald die App ausgeführt wird, können Sie im Hauptmenü die Option "Lokalen Server starten" auswählen. Der Server wird gestartet und eine IP-Adresse wird angezeigt, die andere Spieler verwenden können, um dem Spiel beizutreten.

## Spielen auf einem lokalen Server beitreten

Um einem lokalen Spiel beizutreten, müssen Sie sicherstellen, dass Ihr Gerät mit demselben Netzwerk wie der Host-Computer verbunden ist. Öffnen Sie die ZWOO-App auf Ihrem Gerät und wählen Sie im Hauptmenü die Option "Spiel beitreten". Geben Sie die IP-Adresse des Host-Computers ein, die auf dem Bildschirm des Host-Computers angezeigt wird, und klicken Sie auf "Verbinden". Sie werden dem lokalen Spiel beitreten und können mit dem Spielen beginnen.

## Erweiterte Serverkonfiguration

Die erweiterte Serverkonfiguration ermöglicht es Ihnen, verschiedene Einstellungen für den lokalen Server anzupassen. Dazu gehören die maximale Anzahl der Spieler, die dem Spiel beitreten können, die Spielregeln und andere Optionen. Um auf die erweiterte Serverkonfiguration zuzugreifen, klicken Sie im Hauptmenü der ZWOO-Desktop-App auf "Einstellungen" und dann auf "Serverkonfiguration". Hier können Sie die gewünschten Einstellungen vornehmen und den Server entsprechend anpassen.
